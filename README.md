# theiadocker

*Theia Docker* is the Dockerfile for Eclipse Theia applications on Ubuntu.



### Build

```bash
./build.sh
```



### Deploy

```bash
docker pull craftslab/theiadocker:latest
```



### Run

```bash
./run.sh
```
